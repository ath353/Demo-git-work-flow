# 🔀 Git Workflow Demo

Project nhỏ demo một phần của Git Workflow — quy trình chuẩn giúp các thành viên sử dụng Git làm việc cùng nhau hiệu quả.

---

## 🤔 Git Workflow là gì?

Git Workflow là một công thức / quy trình chuẩn để các thành viên trong team sử dụng Git làm việc cùng nhau.

**Nếu không có Git Workflow, dự án có thể gặp:**

- ⚠️ Code người này đè lên code người kia

- 💥 Code đang lỗi vẫn được đẩy lên nhánh chính khiến ứng dụng sập

- ❓ Không rõ phiên bản nào là ổn định nhất để gửi cho khách hàng

---

## 🎯 Mục đích

Project này được tạo ra để demo và minh họa cách áp dụng Git Workflow trong thực tế.
